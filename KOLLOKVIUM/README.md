11. Tapşırıq: Təsəvvür edin ki, rəqəmsal saat proqramı hazırlayırsınız. İstifadəçidən saat və dəqiqəni tam ədəd kimi daxil etməsini istəyin (məsələn: saat 9, dəqiqə 5). Proqram bu zamanı standart 00:00 formatında çap etməlidir. Yəni, əgər rəqəm tək rəqəmlidirsə, önünə mütləq 0 əlavə edilməlidir.
Nümunə giriş: 9 və 5
Nümunə çıxış: Zaman: 09:05
12. Tapşırıq: Bir mağaza proqramı tərtib edin. İstifadəçi məhsulun qiymətini (float) və aldığı məhsulun sayını (int) daxil etsin. Proqram ümumi məbləği hesablamalı və ekranda "Ödəniləcək məbləğ: X.XX AZN" formatında, yəni qəpiklər mütləq iki rəqəmlə göstərilməklə çap etməlidir.
