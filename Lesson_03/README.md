Mühazirə 3: Giriş və Çıxış Əməliyyatları
1.	Standart giriş-çıxış funksiyaları: printf(), scanf(), fgets(), puts(), getchar(), putchar().
2.	Giriş-çıxış əməliyyatlarında buferləşdirmə (buffering) və buferin ləğv edilməsi (setbuf, fflush).
3.	Giriş daşmalarının (input overflow) təhlükəsiz idarə edilməsi.
4.	Unix/Linux-da faylların yönləndirilməsi (<, >).
5.	Praktika: scanf() yerinə fgets() istifadə edərək təhlükəsiz istifadəçi girişi sisteminin qurulması.
