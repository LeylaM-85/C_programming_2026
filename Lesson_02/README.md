Mühazirə 2: Məlumat Tipləri, Dəyişənlər və Sabitlər
1.	Müxtəlif məlumat tiplərinin yaddaş quruluşu (ölçüsü və diapazonu).
2.	İşarəli (signed) və işarəsiz (unsigned) tiplərlə iş.
3.	const, volatile və enum istifadəsi.
4.	Qeyri-aşkar (implicit) və aşkar (explicit) tip çevrilmələri.
5.	Dəyişənlərin adlandırılması və yaddaş baxımından səmərəli məlumat tipləri üçün ən yaxşı təcrübələr.	Fəsil 2: Dəyişənlər	Səh. 13-15	Dəyişənlərin elanı, yaddaş quruluşu, iki ədədin cəmi (fundamental məntiq).
